Shark Tooth Loans

Project Info:

Shark Tooth Loans is a loan calculator that accepts three values - loan amount, number of payments, and interest rate - and displays a full amortization schedule when the user clicks a button.

Project Details:

  • For:Coder Foundry Bootcamp
  • Technologies:HTML5, CSS3, Bootstrap, JavaScript
