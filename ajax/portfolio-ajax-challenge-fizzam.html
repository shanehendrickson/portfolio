FIZZAM!

Project Info:

FIZZAM! is a classic FizzBuzz project. Using loops and arrays, the app prints out the numbers 1 to 100, substiting any numbers divisible by the chosen values with 'Fizz', 'Buzz', or something else. Each possibility gets its own style. It's FizzBuzz with a little flair.

Project Details:

  • Technologies:HTML5, CSS3, Bootstrap, JavaScript
